# pixem
Source code for the px and (r)em unit converter for style sheets.

## Dependencies
To install the dependencies, run `npm install`.

## Development
```
  $ npm start
```

## Production
```
  $ npm run build
```

## Test

## All tests

```
  $ npm test
```

## Test a directory

```
  $ npm run test src/utils
``` 

## Linting

```
  $ npm run lint
```
